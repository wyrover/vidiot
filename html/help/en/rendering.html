Rendering

Via the menu Sequence → Render settings, the settings of the to be generated video can be changed.

In this dialog you can select the target file name (and container format), the video codec and the audio codec.

Press "Ok" to accept the settings and to continue editing. Press "Ok & render now" to accept the setings and to start rendering the output file immediately. Note that you then can still continue editing on the video. The output video file contains the sequence as how it was when the "Ok" button was pressed.
