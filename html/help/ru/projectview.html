Область проекта

Добавление выбранных файлов с диска

Вы можете добавлять отдельные файлы: для этого щелкните папку правой кнопкой мыши и в контекстном меню выберите пункт "Добавить → Файл(ы) с диска".

Добавление папок с диска

Вместо добавления каждого файла по-отдельности вы можете добавить целую папку с файлами и подпапками. Для этого к контекстном меню выберите пункт "Добавить → Папку с диска". Будет добавлена выбранная папка и ее содержимое.

Добавление или удаление файлов из папки автоматически отразится в области проекта.

Создание папки

Область проекта можно структурировать путем создания папок. Чтобы создать папку, в контекстном меню выберите пункт "Создать → Папку".

Создание видеоряда

Чтобы создать новый пустой видеоряд, в контекстном меню выберите команду "Создать → Видеоряд".

Создание видеоряда с клипами

Чтобы создать видеоряд, заполненный файлами из какой-либо папки, в контекстном меню папки выберите пункт "Создать видеоряд". Все файлы из этой папки будут размещены в новом видеоряде.

Если все файлы имеют похожие имена, в видеоряде можно добавить паузы между клипами на основе различий имен. Например, если все клипы имеют названия типа "scene'20120704 16.02.24.avi", то при настройке параметра длины префикса равным 14 паузы будут созданы между клипами, снятыми в разные даты.
