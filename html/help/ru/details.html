Свойства

В области свойств можно изменять параметры выбранного клипа. Здесь также отображается ход выполнения текущей операции редактирования (например, обрезки).

Свойства клипа

В области свойств клипа можно изменять следующее:
  • Предустановленные значения длительности клипа.
    Значения кнопок можно настроить в файле конфигурации: Справка → Файл конфигурации, значение "LengthButtonsTimesInMilliseconds")
  • Скорость видеоклипа. Скорость аудиоклипов (пока) изменять нельзя.
    Скорость видео можно изменять только если на других дорожках нет других видеоклипов с той же частотой кадров, какая будет у изменяемого клипа.
    Начальный и конечный кадры видео по возможности не подвергаются изменению, хотя это не всегда бывает возможно (например, при увеличении скорости).
  • Непрозрачность клипа. В видеорядах с несколькими дорожками при снижении непрозрачности клипа становятся видимыми клипы на "нижних" дорожках.
  • Масшабирование клипа. Используйте предустановленные значения масштаба или выберите "Задать", чтобы ввести собственный коэффициент масштабирования.
    • Показать все
      Клип будет отображен полностью в конечном видео. Могут появиться черные полосы.
    • Заполнить
      Клип заполнит конечное видео, часть изображения может быть обрезана.
    • Исходный размер
      Будет сохранен исходный размер клипа.
    • Задать
      Коэффициент масштабирования можно задать вручную. Этот метод выбирается автоматически при ручной регулировке значения масштабирования.
  • Выравнивание. Можно выбрать предустановленные параметры выравнивания или задать их вручную.
    • По центру
      Клип в выходном видео будет расположен по центру.
    • По центру по горизонтали
      Клип в выходном видео будет расположен по центру, но только по горизонтали.
    • По центру по вертикали
      Клип в выходном видео будет расположен по центру, но только по вертикали.
    • Задать
      Ввод координат X и Y вручную.
  • Уровень громкости аудиоклипов можно изменять от 0 до 100.

Настройка обрезки

При выполнении обрезки в области свойств отображается исходная и конечная длительности (результат обрезки) выбранного клипа.
