Проекты

Создание проекта

Чтобы создать новый проект, в меню "Файл" выберите пункт "Добавить". откроется окно мастера для создания нового проекта. Новый созданный проект появляется в области проекта с автоматически назначенным именем.

Переименование проекта

Чтобы переименовать проект, в меню "Файл" выберите пункт "Сохранить как". Проект будет сохранен в файл <любое_имя>.vid. После сохранения имя проекта в области проекта будет изменено на введенное вами.

Создание проекта перетаскиванием файлов

Если не открыт ни один проект, перетаскивание файлов или папок из файловой системы на окно программы создаст новый проект. При перетаскивании папки будет создан видеоряд со всеми файлами из папки. При перетаскивании нескольких файлов будет создан видеоряд из этих файлов.

Учтите, что если перетаскиваемые файлы имеют различные частоты кадров и/или частоты дискретизации аудио, видеоряд получит частоту кадров и/или частоту дискретизации, которые имеются у большинства файлов.
