Параметры

Вкладка Название Описание
Общие Загружать последний проект при запуске Если флажок установлен, после запуска программы будет автоматически открыт последний редактировавшийся проект.
Проект Длительность пауз Если значение больше нуля, в создаваемом видеоряде появятся паузы между клипами. Эта настройка определяет длительность таких пауз.
Число символов от начала имени (префикс), которые должны совпадать Количество символов для проверки различий в именах. Если первые символы (префикс) совпадают, пауза не добавляется. В противном случае, если префиксы не совпадают, добавляется пауза.
Видео Частота кадров для новых проектов Выходная частота кадров (в секунду) для проектов. Ее нельзя будет изменить после создания проекта.
Ширина видео по умолчанию Выходная ширина видео для новых проектов. Ее нельзя будет изменить после создания проекта.
Высота видео по умолчанию Выходная высота видео для новых проектов. Ее нельзя будет изменить после создания проекта.
Масштаб видео по умолчанию При добавлении клипов автоматически применяется один из методов масштабирования, который задается здесь. Вы сможете поменять метод масштабирования любого клипа позднее.
Выравнивание видео по умолчанию При добавлении клипов автоматически применяется выравнивание, который задается здесь. Вы сможете поменять выравнивание любого клипа позднее.
Изображения Длительность показа изображений по умолчанию При загрузке изображений автоматически задается длительность их показа. Эта настройка позволяет задать эту длительность.
Аудио Частота дискретизации аудио по умолчанию Выходная частота дискретизации аудио для новых проектов. Ее нельзя будет изменить после создания проекта.
Количество каналов аудио по умолчанию Выходное количество каналов аудио для новых проектов. После создания проекта это значение изменить нельзя.
Шкала Смещение начального маркера (сек.) При выделении нового региона позицию начала можно слегка сместить относительно места, где была нажата клавиша Shift. Используйте отрицательное значение для смещения влево или положительное значение для смещения вправо.
Смещение конечного маркера (сек.) При выделении нового региона позицию конца можно слегка сместить относительно места, где была отпущена клавиша Shift. Используйте отрицательное значение для смещения влево или положительное значение для смещения вправо.
Текст, удаляемый из названий клипов Если клипы имеют похожие имена (например, очень много клипов начинаются с "video_" ...), то вы можете удалить повторяющийся текст из названий клипов, введя его здесь.
