Сочетания клавиш/использование мыши

Тип Действие Описание
Выбор Щелчок левой кнопкой на клипе Выбор/отмена выбора клипа
Alt + щелчок левой кнопкой на клипе Выбор/отмена выбора клипов до конца на всех дорожках
Shift + щелчок левой кнопкой на клипе Выбор/отмена выбора диапазона клипов
Ctrl + щелчок левой кнопкой на клипе Изменение выбора
Ctrl + A Выбор всех клипов
Удаление Delete Удаление выбранных клипов.
Масштабирование Ctrl + колесико мыши Увеличение/уменьшение (работает и при перетаскивании)
= Увеличение
- Уменьшение
Ctrl + = Максимальное увеличение
Ctrl + - Максимальное уменьшение
Прокрутка Shift + колесико мыши Прокрутка шкалы (работает и при перетаскивании)
Перетаскивание правой кнопкой мыши Перемещение по шкале (работает и при перетаскивании)
V Перемещение видимой части шкалы так, чтобы примерно на 1/4 был виден курсор. Работает при воспроизведении.
Перетаскивание Ctrl Изменение точки перетаскивания. Можно использовать так: сперва разместите видеоклип на нужную дорожку, затем, нажав "Ctrl", переместите его к (связанному) аудиоклипу. Затем аудиоклип можно разместить на нужной дорожке.
Shift Удерживайте "Shift", чтобы остальные клипы автоматически освобождали место для перетаскиваемых клипов.
D Временное отключение привязки для текущей операции.
Обрезка Shift Удерживайте "Shift", чтобы освободить место для удлинения клипа или чтобы избежать появления пустого места при сжатии клипа.
D Временное отключение привязки для текущей операции.
U Временная отмена связи между аудио- и видеоклипами при обрезке. Используется для удлинения/сжатия только аудио ИЛИ только видео (иначе изменяется длина аудио И видео одновременно). Подрезка со смещением остальных клипов автоматически отключается.
Редактирование S Разрезает клипы на месте курсора.
Примечание: Эту операцию можно выполнять при воспроизведении, оно будет на время приостановлено, а затем продолжится.
B Разрезает клипы на месте курсора, удаляет первую часть и сдвигает оставшийся фрагмент влево.
Примечание: Эту операцию можно выполнять при воспроизведении, оно будет на время приостановлено, а затем продолжится.
C Удаляет пустой регион под указателем мыши.
E Разрезает клипы на месте курсора, удаляет последнюю часть разрезанных клипов и сдвигает оставшиеся фрагменты вправо.
I Создает нарастание в начале клипа под указателем мыши.
O Создает затухание в конце клипа под указателем мыши.
P Создает переход из предыдущего клипа в клип под указателем мыши.
N Создает переход из клипа под указателем мыши в следующий клип.
1 - 9 Изменение длительности клипа на одно из предустановленных значений, для которых есть кнопки в области свойств.
Курсор Ctrl + ← Перемещение курсора к предыдущему разрезу.
Ctrl + → Перемещение курсора к следующему разрезу.
Ctrl + Home Перемещение курсора в начало.
Ctrl + End Перемещение курсора в конец.
