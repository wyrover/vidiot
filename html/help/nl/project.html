Projecten

Nieuw project

Kies het menu 'Project' en selecteer 'Nieuw' om een ​​nieuw project te maken. Dit zal een venster openen voor het creëren van een nieuw project. Het nieuwe project wordt zichtbaar in het Project overzicht met een gegenereerde naam.

Naam project

Om de project naam te veranderen selecteer 'Project' → 'Opslaan als' om het project op te slaan met een andere naam. Het project wordt opgeslagen als <naam>.vid. Na het opslaan wordt de naam van het project bijgewerkt in het Project overzicht.

Maak project door 'slepen' vanuit het bestandssysteem

Als er geen project is geopend, zorgt het slepen en neerzetten van bestanden/mappen van het bestandssysteem in het maken van een nieuw project. Het project bevat een film die direct alle ondersteunde gesleepte bestanden bevat.

Let op: als er verschillende video frame frequenties en/of audio sample frequenties in de gesleepte bestanden gebruikt worden, dan wordt gekozen voor de meest voorkomende van deze voor het aanmaken van het project.
