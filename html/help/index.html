Vidiot

Welcome to the idiot's 1 guide to Vidiot!

Basic editing with Vidiot consists of the following actions:

  1. Create a new project: File → New
    Change name: File → Save As
    details
  2. Add your video files: Right click on project name in project view → Add → Folder from disk
    details
  3. Create a 'sequence' from the given input files: Right click on folder in project view → Make Sequence
    details
  4. Edit the sequence: Trim, Drag & drop, Copy & Paste, Edit
    details
  5. Select output file and codecs: Sequence → Render settings
    Render the sequence: Sequence → Render
    details
