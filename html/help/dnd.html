Drag and drop from file system

Automatically create projects

If no project is open, dropping files/folders from the filesystem results in creation of a project. All dropped files/folders are included in this project.

If exactly one folder is dropped the initial name of the project equals the folder name.

If exactly one folder is dropped, and that folder contains no subfolders, a sequence is created containing all the files in the folder.

If a list of files is dropped, a sequence containing these files is created.

Note that if there are multiple video frame rates and/or audio sample rates in the files added to the project, then the sequence is not created (because it is then unclear which frame/sample rate to use for the resulting video).

Drag and drop into an existing project

If a project is open, dropped files and/or folders are included into the project view.
