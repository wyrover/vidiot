Projects

Create new project

Select the 'File' menu and then select 'New' to create a new project. The new project becomes visible in the Project View with a generated name.

Directly after creating the project, the Properties dialog is shown. Note: of these settings the video frame rate and the audio sample rate cannot be changed anymore after creating sequences.

Rename project

To change the current name of a project select the 'File' menu and then select 'Save As' to save the project with a different name. The project is saved as <name>.vid. After saving, the project name is updated in the Project View.
