Projects

Create new project

Select the 'File' menu and then select 'New' to create a new project. The new project becomes visible in the Project View with a generated name. The frame rate used in the project is the default frame rate as specified in the Options dialog.

Rename project

To change the current name of a project select the 'File' menu and then select 'Save As' to save the project with a different name. The project is saved as <name>.vid. After saving, the project name is updated in the Project View.
